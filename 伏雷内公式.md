# 伏雷内公式 或 弗勒内-塞雷公式 - R<sup>3</sup>空间上切向、法向和副法向之间的关系

**r** 或 **r**(s) - 曲线，由自然参数描述

**α** 或 **α**(s) - 切向，单位切向量

**β** 或 **β**(s) - 法向 或 主法向，单位主法向量

**γ** 或 **γ**(s) - 副法向，单位副法向量

κ 或 κ(s) - 在s处（自然参数），曲线的曲率

τ 或 τ(s) - 在s处（自然参数），曲线的挠率

$$\frac{d**α**}{ds} = κ(s)**β**$$

$$\frac{d**γ**}{ds} = -τ(s)**β**$$

$$**β** = -κ(s)**α** + τ(s)**γ**$$

注意

$$**α** = \frac{d**r**}{ds} / \vert\frac{d**r**}{ds}\vert$$

$$**β** = \frac{d**α**}{ds} / \vert\frac{d**α**}{ds}\vert$$

$$**γ** = **α** × **β**$$
